This directory must contain virtual directories pointing to core -> ..\..\..\services as well as available services e.g. iqProfile -> ..\..\..\services\IqProfile, iqContacts -> ..\..\..\services\IqContacts, etc.

Alternative, copy files using "copy-schemas.cmd" in "iquomi\services".

Virtual dirs

  core -> \services
  iqAlerts -> \services\iqAlerts
  iqProfile -> \services\iqProfile
  etc.
