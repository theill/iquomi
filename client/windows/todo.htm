Favorites service
  • there's a problem with "FileSystemWatcher". It is case-insensitive which makes it impossible to detect changes for a given file. A solution might be to do a md5 on the folder + file (exclude full-path to "Favorites" system folder) then use this unique id as an id on the uploaded bookmarks. when a change is made we could generate the md5 again and find the key rather fast.
